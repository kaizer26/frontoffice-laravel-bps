# Panduan Penggunaan Aplikasi Front Office (Petugas PST)

Panduan ini disusun untuk membantu petugas Pelayanan Statistik Terpadu (PST) dalam mengoperasikan aplikasi Front Office BPS Kabupaten Tanah Bumbu secara efektif dan efisien.

---

## 1. Persiapan dan Login
- **Tugas Jaga**: Pastikan Anda berada di ruang PST sesuai jadwal yang telah ditentukan.
- **Akses Aplikasi**: Buka browser dan akses alamat: [http://192.168.100.11:8001/login](http://192.168.100.11:8001/login)
- **Kredensial**: Masukkan email dan password Anda. Jika lupa, silakan hubungi administrator.

## 2. Presensi Petugas
Setelah login, Anda wajib melakukan presensi di Dashboard Petugas:
1.  **Clock In**: Klik tombol **"Absen Masuk"** pada widget di bagian atas dashboard.
2.  **Bukti Dukung**: Sesuai kebijakan internal, lakukan pengambilan foto di area PST menggunakan aplikasi **Kipapp** sebagai bukti dukung fisik kehadiran.
3.  **Clock Out**: Jangan lupa klik **"Absen Pulang"** saat jam tugas Anda berakhir.

## 3. Penerimaan Pengunjung (Buku Tamu)
Sambut pengunjung dengan prinsip **5S** (Senyum, Salam, Sapa, Sopan, Santun). Langkah teknis di aplikasi:
1.  Klik tab **"Buku Tamu"**.
2.  Isi identitas lengkap (Nama, Instansi, No. HP, Email).
    - *Tips: Gunakan pencarian otomatis jika pengunjung pernah datang sebelumnya.*
3.  Pilih **Jenis Pelayanan**:
    - **Konsultasi**: Untuk penjelasan teknis atau metadata.
    - **Permintaan Data**: Centang jika pengunjung meminta data spesifik (akan muncul form input nomor surat jika ada).
    - **Pengaduan**: Untuk keluhan atau kritik.
    - **Lainnya**: Rekomendasi Statistik, Perpustakaan, dll.
4.  Pilih **Sarana Kunjungan**: Pilih "Langsung" atau "Online".
5.  Klik **"Simpan Buku Tamu"**.

## 4. Proses Pelayanan & Koordinasi
1.  **Layanan Saya**: Pantau tab ini untuk melihat daftar permintaan yang sedang Anda tangani.
2.  **Koordinasi IP**: Jika membutuhkan jawaban detail dari *Subject Matter*, gunakan fitur **"Kirim Notifikasi ke Grup WA"** di detail layanan. Ini akan mengirimkan template ringkasan keperluan pengunjung ke grup koordinasi internal.
3.  **Drive Data**: Gunakan tombol **"Drive Kumpulan Data"** di tab Layanan Saya untuk mengakses koleksi data yang tersedia di Google Drive.

## 5. Pemenuhan Permintaan Data (Khusus Data)
Untuk layanan permintaan data:
1.  Klik tombol **"Update"** pada layanan terkait.
2.  Ubah status menjadi **"Diproses"** atau **"Siap Diambil"**.
3.  Jika sudah siap, ubah status menjadi **"Selesai"**.
4.  **Nomor Surat Balasan**: Gunakan tombol **"Generate Nomor Surat Balasan"** (Smart Reply) untuk mendapatkan nomor surat balasan otomatis sesuai standar penomoran BPS.
5.  Upload file surat balasan (jika ada) dan tambahkan Tags data untuk memudahkan pencarian di masa depan.

## 6. Closing: Rating & Survei SKD
Setelah pelayanan diberikan secara optimal:
1.  Minta pengunjung mengisi rating. Klik tombol **"Rating"** pada layanan terkait.
2.  Akan muncul pilihan:
    - **QR Code**: Biarkan pengunjung memindai QR Code di layar.
    - **Kirim WA**: Klik ikon WhatsApp untuk mengirimkan link rating langsung ke nomor HP pengunjung.
3.  **Survei SKD**: Khusus untuk Permintaan Data, pastikan pengunjung juga mengisi link SKD yang disediakan (tersedia QR Code dan tombol kirim WA khusus SKD).
4.  Pantau status SKD: Jika status sudah **"SKD Diisi"** (berwarna hijau), maka proses administrasi selesai.

---

> [!IMPORTANT]
> Selalu pastikan status layanan diubah ke **"Selesai"** agar data statistik pelayanan harian/mingguan terekam dengan benar di sistem laporan pimpinan.
